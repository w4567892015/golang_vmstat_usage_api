# Golang Vmstat Usage Api

The linux vmstat to api.

## Install

```shell
wget -q https://github.com/w4567892015/golang_vmstat_usage_api/releases/download/v0.1.0/vmstat_usage_api && chmod +x vmstat_usage_api
```

## Usage

```shell
$ ./vmstat_usage_api
$ curl http://localhost:3000
```
